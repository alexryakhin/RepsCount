Welcome to **RepsCount**, a comprehensive and intelligent workout tracking app designed to help you achieve your fitness goals. Whether you're at the gym, at home, or on the go, RepsCount provides everything you need to track your progress and stay motivated.

## üöÄ Current Features

### 1. **Comprehensive Workout Management**
- **Workout Templates**: Create, edit, and manage custom workout templates
- **Exercise Library**: 150+ exercises across all muscle groups with detailed categorization
- **Set Tracking**: Track reps, weight, and time for each exercise set
- **Workout Scheduling**: Schedule workouts with calendar integration using EventKit
- **Progress Tracking**: Track completion status and workout duration
- **Location Tracking**: Optional GPS location saving for workouts

### 2. **Advanced Exercise System**
- **Muscle Map Visualization**: Interactive muscle group highlighting showing primary and secondary muscle groups
- **Exercise Categories**: Organized by muscle groups, equipment, and exercise type
- **Exercise Details**: Track notes, default sets/reps, and personal records
- **Equipment Filtering**: Filter exercises by available equipment

### 3. **Smart Data Management**
- **iCloud Sync**: CloudKit integration for seamless cross-device synchronization
- **Core Data**: Robust local data persistence with automatic conflict resolution
- **Calendar Integration**: Add workout events to your system calendar
- **Analytics**: Firebase Analytics for user behavior tracking

### 4. **User Experience**
- **Pure SwiftUI Architecture**: Modern, responsive UI with NavigationStack
- **Multi-language Support**: 6 languages (English, German, Spanish, French, Croatian, Russian)
- **Dark/Light Mode**: Full system theme support
- **Haptic Feedback**: Tactile responses for better UX
- **Confetti Animations**: Celebration effects for achievements
- **Onboarding**: Guided first-time user experience

### 5. **Technical Excellence**
- **MVVM Pattern**: Clean separation of concerns
- **Combine Framework**: Reactive data flow and state management
- **Service Manager**: Centralized dependency injection
- **Error Handling**: Comprehensive error management system
- **Performance Optimizations**: Lazy loading and efficient data structures
